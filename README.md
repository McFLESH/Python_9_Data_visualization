С помощью matplotlib можно настроить внешний вид графиков, добавить подписи, легенды, аннотации.


- Импорт библиотек
- import pandas as pd
- import matplotlib.pyplot as plt
 
- Чтение данных из файла data.csv
- df = pd.read_csv('data.csv')
 
- Названия колонок, которые необходимо отобразить: 'Product', 'Sales'
- xdata = df['Product']
- ydata = df['Sales']
 
- Построение диаграммы типа “bar” (столбчатая)
- plt.bar(xdata, ydata)
 
- Вызов отображения диаграммы
- plt.show()


Создадим простой линейный график с помощью matplotlib:


- import matplotlib.pyplot as plt
 
- Пример данных
- x = [1, 2, 3, 4, 5]
- y = [2, 4, 6, 8, 10]
 
- Cоздаем пространство и оси
- fig, ax = plt.subplots()
 
- Строим график данных
- ax.plot(x, y)
 
- Добавляем метки осей и заголовок
- ax.set_xlabel('Ось X')
- ax.set_ylabel('Ось Y')
- ax.set_title('Линейный график')
 
- Выводим график на экран
- plt.show()


matplotlib позволяет сравнивать данные на одном графике или отображать несколько графиков рядом с помощью plt.subplot() и plt.subplots().

Функция subplot() имеет три обязательных параметра: количество строк в графике; количество столбцов в графике; номер ячейки, куда будут выводиться графики после вызова функции. Ячейки нумеруются построчно, начиная с 1.
